# Background Images

This directory should contain three background images:

1. `hero-bg.webp` - Background for the Hero section
2. `services-bg.webp` - Background for the Services Overview section  
3. `contact-bg.webp` - Background for the Contact Us section

## Image Requirements

- Format: WebP (with PNG fallbacks if needed)
- Recommended size: 1920x1080 or higher
- Aspect ratio: 16:9 or similar
- Style: Dark, elegant, minimal aesthetic matching the Anvari Equestrian theme
- File size: Optimized for web (under 500KB per image recommended)

## Adding Images

1. Place your WebP images in this directory with the exact filenames listed above
2. If using PNG fallbacks, name them `hero-bg.png`, `services-bg.png`, `contact-bg.png`
3. Update the `backgroundImages` array in `src/App.tsx` if using different filenames
